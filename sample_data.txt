Streamlit은 데이터 스크립트를 단 몇 분 만에 공유 가능한 웹 앱으로 바꿔주는 오픈 소스 Python 라이브러리입니다.
전체 머신러닝 및 데이터 과학 팀을 위해 구축되었습니다.
프론트엔드 개발 경험이 없어도 Python만으로 아름다운 UI를 만들 수 있습니다.
st.write(), st.dataframe(), st.chart() 등의 직관적인 API를 제공합니다.
RAG(Retrieval-Augmented Generation)는 대규모 언어 모델(LLM)의 출력을 최적화하여 응답이 신뢰할 수 있는 외부 지식 베이스를 참조하도록 하는 기술입니다.
이 챗봇의 이름은 "Moa(모아)"입니다. 귀여운 다람쥐 캐릭터를 모티브로 만들어졌습니다.
사용자에게 항상 친절하고 명랑하게 대답하며, 문장 끝에 다람쥐 이모티콘(🐿️)을 자주 사용합니다.
이 챗봇은 OpenAI의 GPT 모델과 ChromaDB를 사용하여 구현되었습니다.
사용자의 질문과 가장 유사한 문서를 ChromaDB에서 검색하여 답변 생성 시 참조합니다.
API 키는 보안을 위해 .env 파일에 보관해야 합니다.
